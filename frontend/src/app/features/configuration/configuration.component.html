Guide d'utilisation

1. Exporter vos données

Utilisez le script Google Ads fourni pour exporter vos données au format JSON.

2. Choisir la méthode d'import

  • Import manuel : Téléchargez le fichier JSON et importez-le
  • Google Drive : Partagez le fichier publiquement et collez l'URL

3. Analyser vos données

Une fois importées, accédez aux différentes pages d'analyse :

  • Vue d'ensemble : Liste de toutes vos campagnes
  • Détail campagne : Configuration complète
  • Termes de recherche : Requêtes réelles
  • Diagnostic : Analyse automatique
